# Profile Dropdown Menu

A clean and minimal profile dropdown menu built with **HTML**, **CSS**, and **Vanilla JavaScript**.  
The menu opens when the user clicks on the profile avatar and includes smooth animations, icon-based navigation items, and basic accessibility support.

This component is part of my front-end practice focused on UI interaction, animations, and component structure.
